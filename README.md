# Rock Paper Scissors
This project is a CLI-based Rock Paper Scissors game built with Python.
I plan to expand this project as much as I can over time.

---

## Brief History

Rock Paper Scissors originated in **China during the Han Dynasty** (206 BCE – 220 CE) under the name "shoushiling." It spread to **Japan**, where it evolved into "**jan-ken**" — the direct source of the modern version. The game reached the Western world in the early 20th century through Japanese contact and has since become a global standard for decision-making.[Wikipedia](https://en.wikipedia.org/wiki/Rock_paper_scissors)

## Rules

- First player to reach **3 points** wins the match
- Each round winner gets **1 point**
- Standard rules:
  - ✊ Rock beats Scissors
  - ✋ Paper beats Rock  
  - ✌️ Scissors beats Paper
  - Same choice = tie (no points)
